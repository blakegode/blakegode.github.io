My Projects

Project Title 1

This project trains a machine learning model to predict how many points NBA players would score in a given game. The project first cleans and prepares data of NBA box scores from the 2023-2024 season. The clean data is then entered into Scikit Learn's Sequential Feature Selection API which selects 30 features that contain the most corelation to the target value. Finally the data is used to train Scikit Learn's Ridge Classifier model which is used to predict how many points each NBA player will score in the next game.

Brief description of the project.
